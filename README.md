# Meta-Source Ledger

> A cross-domain creative-technical framework exploring identity systems, cipher visualization, mythology, and generative platforms.

## Overview

This repository contains comprehensive research and specifications for a modular rendering pipeline that visualizes encoding/cryptographic systems through unified conceptual frameworks—from Matrix-style digital cascades to Zodiac symbolic ciphers to WWII Enigma mechanics.

## Features

- **Identity Systems** — Temporal identity modeling, visualization algorithms, ethics frameworks
- **Cipher Visualization** — Visual cryptanalysis, generative art rendering, WebGL pipelines
- **Mythology & Symbolism** — Golden ratio protocols, numerology synthesis, symbolic languages
- **Platform Integration** — Unified architecture, component libraries, multiverse frameworks

- **Visualization:** p5.js, Three.js, WebGL shaders
- **Frontend:** React 18+, TypeScript
- **Patterns:** State machine paradigm, modular cipher rules, renderer abstraction
